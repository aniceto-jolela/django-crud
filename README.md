# Django authentication crud

A simple user crud.
 
== Version
- pip: ``24.0``
- django: ```5.1.2```
- db: ``sqlite3``
- django-crispy-forms: ``2.3``
- crispy-bootstrap4: ``2024.10``
- Pillow: ``11.0.0``
###
- - install Pillow
```shell
  python -m pip install Pillow
```
or
```shell
  pip install Pillow
```

forms

